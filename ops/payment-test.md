# Payment Test

## Safe PayPal button test

1. Replace `PAYPAL_LINK_HERE` in `landing-page/index.html` with your public PayPal.me or checkout URL.
2. Open `landing-page/index.html`.
3. Click "Pay $5 with PayPal".
4. Confirm the page opens PayPal and shows the correct recipient and amount if your link includes an amount.
5. Do not complete a payment to yourself unless you intentionally want to test with a separate account.
6. Return to the landing page and confirm the contact instructions are clear.

## If using PayPal.me

Common public format:

```text
https://www.paypal.me/YOURNAME/5USD
```

Only use your own public PayPal.me link.

## If using PayPal checkout URL

Use the public buyer-facing checkout URL. Do not paste dashboard URLs, API keys, client secrets, or private account settings.

## If no PayPal link is ready

Use this temporary message in posts:

```text
DM me and I’ll send the PayPal link manually.
```

Then send only your public PayPal payment URL.
